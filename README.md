## Letters and Numbers

This is a simple application written in Cursor to test out agentic coding. The application contains a list of words, and you can increase or decrease the number of letters on the word - and when you do, the results area will display words that contain that many letters. For example, if you let the number of letters to 3, then words like "cat" or "hen" will appear in the list.

### Installation
1. Copy/rename the `.env.example` file to `.env`
2. Create a .sqlite file in the /database folder and call it `database.sql`
